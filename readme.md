Bulk Assessments

C# Wrappers around a sequences of transactions with Gemini GenAI and Flash AI Models using the Google.GenAI namespaces. This implementation uploads Rubric and Report files to the namespace's cloud URIs. For each Report, create a Score Workbook from an existing Template file. Collect three assessments and insert them into the Workbook. Json-formatted response conventions are requested but sometimes the exact Schema is still being ignored, especially if using Flash-Lite models. An extra-slow implementation, copious use of time delays. Detailed Console logging.
